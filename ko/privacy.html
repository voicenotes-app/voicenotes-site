개인정보 처리방침

최종 업데이트: 2026년 4월 1일

요약

VoiceNotes는 귀하의 데이터를 외부 서버에 수집, 전송 또는 저장하지 않습니다. 모든 데이터는 귀하의 기기에 보관됩니다.

1. 수집하지 않는 데이터

  • 오디오 녹음
  • 전사 텍스트
  • 회의록
  • 채팅 메시지
  • 사전 항목
  • 사용 분석 또는 텔레메트리
  • 기기 정보
  • IP 주소

2. 로컬 처리

모든 오디오 전사(whisper.cpp 사용) 및 AI 처리(Ollama 사용)는 전적으로 귀하의 기기에서 로컬로 수행됩니다. 오디오, 텍스트 또는 AI 쿼리가 외부 서버로 전송되지 않습니다.

3. 네트워크 통신

VoiceNotes가 인터넷에 연결되는 경우는 다음과 같습니다:

  • 라이선스 활성화: 결제 제공업체(Polar)를 통한 라이선스 키의 일회성 확인. 라이선스 키만 전송되며 오디오나 텍스트 데이터는 전송되지 않습니다.
  • 라이선스 검증: 정기적인 재확인(온라인 시 24시간마다). 오프라인 상태에서 최대 30일간 작동합니다.
  • 모델 다운로드: Whisper 모델은 HuggingFace에서 다운로드됩니다. Ollama 모델은 Ollama 서버를 통해 다운로드됩니다. 이는 사용자가 직접 시작하는 작업입니다.
  • 업데이트 확인: 자동 업데이트가 활성화된 경우 본 앱이 새 버전을 확인합니다.

4. 데이터 저장

귀하의 모든 데이터는 기기에 로컬로 저장됩니다:

  • 데이터베이스: 시스템 앱 데이터 폴더의 SQLite 파일
  • 오디오/비디오 파일: 문서 폴더 내
  • AI 모델: 앱 캐시 폴더 내

본 앱을 제거하고 데이터 폴더를 삭제하면 언제든지 모든 데이터를 삭제할 수 있습니다.

5. 서드파티 서비스

  • Polar (polar.sh): 결제 처리 및 라이선스 키 발급을 담당합니다. Polar 이용약관이 적용됩니다.
  • Ollama: 귀하의 기기에서 로컬로 실행됩니다. 모델 다운로드 시를 제외하고 Ollama 서버로 데이터가 전송되지 않습니다.
  • HuggingFace: Whisper 모델은 HuggingFace 서버에서 다운로드됩니다.

6. 아동 개인정보 보호

VoiceNotes는 13세 미만의 아동을 대상으로 하지 않습니다. 아동의 정보를 고의로 수집하지 않습니다.

7. 본 방침의 변경

본 개인정보 처리방침은 수시로 업데이트될 수 있습니다. 최신 버전은 항상 voice-notes.org/privacy에서 확인할 수 있습니다.

8. 문의

본 개인정보 처리방침에 관한 질문은 voice-notes.org를 방문해 주십시오.
